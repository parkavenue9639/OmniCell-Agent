# Graph A/B 核心行为基线

本目录保存 Graph A/B 必须持续通过的可复现核心行为基线。基线只锁定领域输入输出、路由、反馈循环、聚合、评分和 marker contract 语义，不冻结内部节点拓扑、旧模块入口、提示词全文、并发完成顺序或 trace 时间戳。

验证分为三层：

1. 确定性契约：对结构化状态、路由、评分、合并和报告投影做精确比较。
2. 受控模型替身：使用固定结构化响应验证完整 Graph A/B 数据流，不访问真实模型或 Docker。
3. 真实模型观察：仅作为可选的科学输出分布证据，不进入默认核心行为门槛。

默认测试必须离线、可重复，并且只在临时目录写入 fixture 或 artifact。旧模块路径、旧类名、旧函数、旧 CLI 和固定 DAG 入口不属于本目录的保护对象。
